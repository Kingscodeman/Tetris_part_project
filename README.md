# 🧱 Java 俄羅斯方塊 (Java Tetris)

這是一個使用 Java 與 Swing 框架從零打造的經典俄羅斯方塊遊戲。專案不僅實作了俄羅斯方塊的核心玩法，還加入了現代化 Tetris 遊戲常見的輔助功能以及充滿復古氣息的 MIDI 音效。

本專案同時也是一套階段式開發的教學/展示專案，非常適合用來學習 Java 物件導向程式設計 (OOP)、圖形使用者介面 (GUI) 與基本遊戲迴圈 (Game Loop) 邏輯。

---

## ✨ 遊戲特色 (Features)

* **經典核心玩法**：完整的方塊隨機生成、自動下落、碰撞偵測與滿列消除機制。
* **影子方塊 (Ghost Piece)**：在畫面底部即時顯示方塊落點的透明陰影，幫助玩家精準定位。
* **預覽看板 (Next Piece Preview)**：介面右上方會顯示下一個即將出現的方塊，方便玩家規劃策略。
* **遊戲狀態控制**：支援遊戲暫停與 Game Over 狀態判定。
* **復古遊戲音效**：內建 SoundManager 控制 MIDI 事件，為方塊操作與消除加入電子音效，提升打擊感與遊戲體驗。
* **計分系統**：隨著消除的行數累積，獲得相應的遊戲分數。

## 🎮 如何遊玩 (How to Play)

啟動遊戲後，點擊遊戲視窗即可開始遊玩。請使用以下鍵盤按鍵來控制：

| 按鍵 (Key)   | 動作 (Action)       | 說明 |
| :---         | :---                | :--- |
| `←` 左方向鍵 | **左移**            | 將方塊向左移動一格 |
| `→` 右方向鍵 | **右移**            | 將方塊向右移動一格 |
| `↑` 上方向鍵 | **旋轉**            | 順時針旋轉方塊 |
| `↓` 下方向鍵 | **加速下落 (Soft Drop)** | 加快方塊落下的速度 |
| `空白鍵 (Space)` | **直接落下 (Hard Drop)**| 瞬間將方塊重擊至底部 |
| `P` | **暫停 (Pause)** | 暫停/繼續遊戲 |

## 🚀 快速啟動 (How to Run)

本遊戲以原生 Java 開發，不依賴外部龐大的 Engine，請確認您的電腦已安裝 JDK (Java Development Kit)，接著在終端機 (Terminal) 中依序執行以下指令：

**1. 編譯原始碼：**
```bash
javac -encoding UTF-8 *.java
```

**2. 執行遊戲：**
```bash
java Tetris
```

## 📁 核心程式碼簡介 (Source Code Structure)

如果您想直接閱讀原始碼，以下是遊戲的主要檔案模組化架構：

* **`Tetris.java`**：遊戲的主要入口點 (Main Class)，負責建立繪製視窗 (JFrame)。
* **`TetrisPanel.java`**：遊戲核心引擎 (JPanel)。負責處理遊戲迴圈 (Timer)、鍵盤事件、陣列碰撞邏輯、畫面渲染，以及管理遊戲狀態。
* **`Tetromino.java`**：操作單個方塊的實體物件。管理座標、旋轉狀態與行動邏輯。
* **`TetrominoType.java`**：定義方塊類型的列舉 (Enum)。包含 7 種經典形狀的陣列資料與對應顏色。
* **`SoundManager.java`**：音效管理器。透過 Java 內建的 MIDI 系統控制遊戲音效。
